# Figma-to-UX Review

AI UX review agent for screenshots, Figma exports and interfaces.

## Features

- UX analysis
- Accessibility review
- CRO recommendations
- UI consistency analysis
- Design system review
- Screenshot analysis
- Product scoring

## Stack

- Python
- OpenAI API
- Vision analysis
- dotenv
- GitHub

## Usage

```bash
python main.py --image samples/interface.png
```

```bash
python main.py --text "Modern SaaS dashboard"
```
